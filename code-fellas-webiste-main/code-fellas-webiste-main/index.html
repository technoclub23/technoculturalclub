About

Techno Cultural Club is organizing its First Event - Code Fellas Hackathon for the GBU students from 13th Dec to 15th Dec, 2021 in School of Information and Communication Technology (SOICT). A perfect opportunity for students to showcase their Skills through their Innovative Projects .

Themes

  • Web Development

    Static and dynamic website to solve problems creatively. Including aspects such as web design, web programming, and database management.

  • App Development

    Developing useful applications for mobile devices. Using modern tools to create a good android experience for users.

  • Block Chain

    Creating a system of recording information in a way that makes it difficult to change, hack or cheat the system. Blockchain is a growing list of records, called blocks, that are linked together using cryptography

  • Machine Learning

    Solving real world problems by using optimized and effective ML algorithms. Making machine mimic humans by training them with past data and outcomes.

  • Internet Of Things

    Establishing a network of physical objects for connecting and exchanging the data with other devices and systems over the internet. Embedding the system by sensors, processing ability, software and other technologies to develop a network over systems.

  • Open Innovation

    Any project based on innovative ideas or technologies to solve real time problems. Open innovation includes projects on any technology such as Data Structures and Algorithms, python, go lang, matlab or any other programming language.

Rules:

  • Registration Fees Rs 50/- per team.
  • Team Size : 2-3 Members (max : 3).
  • Problem Statement will be chosen by your own.
  • Project Submissions will be done online .
  • Attendance will be given to those who will Register and Submit their Projects on time.

Judging Criteria:

  • Problem statement - 10 Marks
  • Documentation(Goals, Scope etc) - 15 marks
  • Accuracy,Time complexities, Space Complexity - 20 Marks
  • Technologies covered : 10 Marks
  • Security, Creativity- 15 Marks
  • Extent to which problem solved - 10 Marks
  • Video and PPT - 20 marks
