About

It's a 3 hour long coding challenge conducted by Techno Cultural Club in collaboration with Microsoft Learn Student Ambassador.
It also includes a tech session on Low Code/ No Code.
Top three winners will be selected based on the rank list.

Prerequisites expected from participants:-
1. Good Knowledge of one the following programming languages - C++, JAVA, PYTHON.
2. Good Knowledge of Concepts of Data Structures and Algorithms.
3. A basic knowledge of dynamic programming will be a plus.
